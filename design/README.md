# gitpull Design Package

This folder contains the design package for the Android app `gitpull`.

## Files

- `DESIGN_SYSTEM.md`: readable design system and implementation guidance.
- `spec/gitpull-ui-spec.json`: structured UI spec with tokens, components, screens, states, and interactions.
- `images/01-colors-typography.png`: color and type board.
- `images/02-layout-spacing.png`: layout and spacing board.
- `images/03-controls-states.png`: controls and interaction states board.
- `images/04-cards-lists.png`: cards and PDF list board.
- `images/05-user-flow.png`: user flow board.
- `images/06-apk-logo.png`: APK logo and adaptive icon direction.

## Product Summary

`gitpull` manually refreshes a local Android Obsidian vault folder from a GitHub repository snapshot. It is pull-only, treats GitHub as the source of truth, and gives quick access to pulled PDFs through Android's open-with flow.
