# gitpull Logo Candidates

These are SVG logo candidates sized for app-icon use. Each SVG uses a `1024 x 1024` viewBox and keeps the important mark inside the circular launcher safe area.

## Candidates

- `gitpull-logo-a-vault-arrow.svg`: folder/vault plus pull arrow, closest to the current app UI.
- `gitpull-logo-b-branch-drop.svg`: Git branch nodes flowing into a vault card.
- `gitpull-logo-c-gp-arrow.svg`: monogram-style mark, strongest at small sizes.
- `gitpull-logo-d-minimal-safe.svg`: simplest high-contrast icon with the most mask padding.

## Preview

Open `logo-preview.html` to compare square and circular masks at `192`, `96`, `48`, and `24` px. The generated preview image is `logo-preview-sheet.png`.
