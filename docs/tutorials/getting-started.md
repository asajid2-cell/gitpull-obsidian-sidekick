# Getting Started

This tutorial refreshes an Obsidian vault folder on Android from a GitHub repository.

## Before You Start

You need:

- The `gitpull` APK installed.
- Obsidian Mobile installed.
- A GitHub repository containing the vault files.
- A GitHub token if the repository is private or if you want repository browsing.

## Steps

1. Open `gitpull`.
2. Paste a GitHub token if you want to browse private/account repositories.
3. Tap `Load` in the GitHub repositories card.
4. Tap `Use` beside the repository that contains your vault.
5. Confirm the branch.
6. Choose the Android folder where the vault should live.
7. Tap `Save`.
8. Open the Pull tab.
9. Tap `Pull`.
10. Open the same folder in Obsidian Mobile as an existing vault.

After the first setup, normal use is just opening `gitpull` and tapping `Pull`.

For a screen-by-screen walkthrough, see `visual-walkthrough.md`.
