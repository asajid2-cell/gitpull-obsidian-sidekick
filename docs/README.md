# Source Documentation

This branch contains the Android source, tests, design package, and project notes for `gitpull`.

Use the `main` branch for the release package and APK. Use this branch when you want to inspect or build the app.

## Contents

- `app/`: Android application source and tests.
- `design/`: UI design system, generated design boards, and JSON UI spec.
- `docs/validation/VALIDATION_SUMMARY.md`: validation summary.
- `docs/planning/RELEASE_SCOPE.md`: implementation scope.
- `docs/tutorials/visual-walkthrough.md`: Pixel 9 Pro XL screenshots for setup, pull, PDFs, and settings.
- `docs/assets/screenshots/`: walkthrough screenshots captured from the app.

Generated emulator dumps, third-party APKs, local machine paths, and scratch files are not included.
