# Refresh A Vault

Use this when the Android folder already exists and the app has been configured.

1. Open `gitpull`.
2. Check that the Pull tab shows the expected repository and folder.
3. Tap `Pull`.
4. Wait for `Snapshot ready`.
5. Open Obsidian Mobile and use the same folder.

If a pull fails, `gitpull` keeps the previous usable snapshot instead of replacing it with partial files.
