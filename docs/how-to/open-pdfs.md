# Open Pulled PDFs

`gitpull` indexes PDFs after each successful pull.

1. Pull the vault.
2. Open the PDFs tab.
3. Use search if the list is long.
4. Tap a PDF row.
5. Choose the Android app you want to use for reading or annotation.

The app uses Android's standard open-with flow. Any installed PDF reader or annotation app can appear in the chooser.
