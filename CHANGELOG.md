# Changelog

## 0.1.0 - 2026-05-14

- Added the first Android APK release for `gitpull`.
- Added GitHub token support for repository browsing.
- Added manual Pull flow for refreshing a chosen Android folder.
- Added ZIP snapshot extraction with root stripping and zip-slip protection.
- Added snapshot replacement behavior that preserves the previous snapshot after failed pulls.
- Added PDF indexing and Android open-with handoff.
- Added Obsidian Mobile folder/vault compatibility validation.
- Added private GitHub fixture validation against a controlled vault fixture.
